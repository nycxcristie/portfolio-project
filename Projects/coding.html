Coding Projects

This is my very first coding project I did while taking the "SheCodes Basics" workshop. In this course, I learned how to use HTML, CSS, Javascript, and the VSCode Editor to create a single page website - about Bananas.
After dipping my feet gently into the world of coding, I was motivated to learn more. In July of 2022, I created this Vanilla Weather App with the tools and techniques I learned in the "SheCodes Plus" workshop. In this workshop, I continued to use Visual Studio Code Editor to code. After a quick review of HTML and CSS, we dove into Javascript which entailed understanding strings, arrays, loops, objects and functions. I also learned about APIs and how to use Bootstrap for coding. During this course, I figured out how to set up a Netlify account for hosting and GitHub for committing and pushing my codes into a repository. My favorite part was drawing a wireframe and then turning the image into HTML/CSS code.
This current website, is my third coding project which entails a history of my most recent work A.K.A. My Portfolio. It is a project I did with SheCodes as well. "SheCodes Responsive" was a bit intimidating to start after going through "SheCodes Plus", however it turned out better than expected. I thoroughly enjoyed designing this website with all that I've learned so far. I feel much more confident with using VS Code, GitHub and Netlify, not to mention writing HTML and CSS codes in general. Making a webpage responsive is a little tricky, but I had a lot of fun figuring it out with a little help from the internet and some brain power.
